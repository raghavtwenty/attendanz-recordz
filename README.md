# Attendanz Recordz
# Online Attendance Calculator

Looking for an easy way to take online class attendance, then this is the right app for you...

Programmed in python using tkinter GUI library.

This program takes total students strength as excel file format input and students present for now as another input and finds out the absentees.

Before running the .py file install the libraries mentioned in the requirements file.
